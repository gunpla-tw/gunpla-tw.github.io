gunpla-tw.github.io
===================

gunpla-tw public site.

Contribute & Publish
-------------------
* Make sure you're the group memeber of gunpla-tw

### Clone this repo
```shell
git clone git@github.com:gunpla-tw/gunpla-tw.github.io.git
```

### Prepare
```shell
bundle install
```

### Local Server
```shell
jekyll serve --watch
```

### Publish your change

```shell
rake publish
```
